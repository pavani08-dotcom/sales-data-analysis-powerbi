# Power BI Dashboard Setup Guide

This guide will walk you through creating an interactive Sales Data Analysis Dashboard in Power BI using the `cleaned_sales_data.csv` generated by the Python script.

## 1. Import Data into Power BI
1. Open Power BI Desktop.
2. Click on **Get Data** -> **Text/CSV**.
3. Navigate to your project folder and select `cleaned_sales_data.csv`.
4. Click **Load** (or **Transform Data** if you wish to verify the data types in Power Query, though the Python script has already cleaned it).

## 2. Verify Data Types
Ensure your columns have the correct data types:
- `Date`: Date
- `Quantity`: Whole Number
- `UnitPrice`: Decimal Number
- `TotalRevenue`: Decimal Number / Currency
- Everything else should be Text.

## 3. Create DAX Measures (Optional but Recommended)
To make your dashboard dynamic, create the following measures. Right-click on your dataset in the Fields pane and select **New Measure**:

- **Total Revenue**: `Total Revenue = SUM(cleaned_sales_data[TotalRevenue])`
- **Total Orders**: `Total Orders = DISTINCTCOUNT(cleaned_sales_data[OrderID])`
- **Total Items Sold**: `Total Items Sold = SUM(cleaned_sales_data[Quantity])`
- **Average Order Value**: `AOV = [Total Revenue] / [Total Orders]`

## 4. Build the Visuals

### Visual 1: KPI Cards (Top Metrics)
- **Visual Type**: Card (Create 4 separate cards)
- **Fields**: Drag the measures you created (`Total Revenue`, `Total Orders`, `Total Items Sold`, `AOV`) into each card.
- **Placement**: Place these along the top of your dashboard.

### Visual 2: Sales Trend over Time
- **Visual Type**: Line Chart
- **X-axis**: `Date` (You can use the Date Hierarchy to show Year and Month)
- **Y-axis**: `Total Revenue`
- **Purpose**: Shows the seasonality and overall trend of sales.

### Visual 3: Revenue by Category
- **Visual Type**: Donut Chart or Bar Chart
- **Legend / Y-axis**: `Category`
- **Values / X-axis**: `Total Revenue`
- **Purpose**: Highlights which product categories bring in the most money.

### Visual 4: Top 10 Products by Quantity Sold
- **Visual Type**: Clustered Bar Chart
- **Y-axis**: `Product`
- **X-axis**: `Quantity` (Sum)
- **Filter**: Open the Filters pane, select the `Product` filter, change Filter Type to "Top N", set it to Top 10 By Value `Quantity`, and click Apply Filter.

### Visual 5: Customer Behavior (Orders per Category)
- **Visual Type**: Matrix or Table
- **Rows**: `CustomerID`
- **Values**: `Total Orders`, `Total Revenue`
- **Sort**: Sort by `Total Revenue` descending to see your best customers.

## 5. Add Interactivity and Polish
- Add a **Slicer** for the `Category` column so users can filter the entire dashboard by specific categories.
- Add a **Slicer** for the `Date` column (using the "Between" style) so users can focus on specific time periods.
- Change the Theme (View -> Themes) to a professional color palette.
- Add a Text Box at the top for the title: "Sales Data Analysis Dashboard".

You now have a fully functional, interactive Power BI dashboard ready to be presented!
